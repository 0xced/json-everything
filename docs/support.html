Support

If you find you need help with any of the json-everything libraries you have several options:

  • Join the my Slack workspace.
  • Post a question on StackOverflow*, or
  • Email me at my GitHub username at yahoo.

For feature requests, bug reports, or documentation errata reports, please feel free to create an issue.

* If you post a question on StackOverflow, please let me know via one of the other methods. This library collection is still rather new and SO tags haven't been created yet.
